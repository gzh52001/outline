# Hybrid

## app类型
* WebApp
    * 技术栈：html+css+js
    * 优点：
        * 成本低
        * 跨平台
    * 缺点
        * 不能调用硬件设备
* NativeApp
    * android
        * 技术栈
            * xml
            * java
    * iOS
        * 技术栈
            * xml
            * Object-C/swift

* HybridApp
    > 综合了WebApp与NativeApp的优点，采用半web半Native的开发模式
    * Native主导
    * H5主导
